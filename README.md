# AluraBooks-Estudos-Alura
https://aebombom.github.io/AluraBooks-Estudos-Alura/
# Sobre o Projeto 
O projeto AluraBooks é uma atividade da plataforma de cursos Alura com objetivo o aprendizado e a pratica na criação de uma interface responsiva, criando um interface que simula um site de divulgação e vendas e de livros relacionados a programação, tecnologia e desenvolvimento, usando __HTML e CSS__ como base.
Esse projeto tem como foco criar um site que possa funcionar em diferentes dispositivos, sem que isso afete a experecia do usuario.
# Estrutura e conteúdo 
O Projeto AluraBooks segue uma estrutura de __mobile first__.
